# Telecom Plans Comparison

## The Data

The work based on the toy data provided by Yandex.Practicum. It comes in five separate data sets:
#### users
- first name
- last name
- age
- subscription date
- churn date
- city
- plan name
#### calls
- call date
- duration
- user id
#### messages
- message date
- user id
#### internet
- megabytes spent while in the session
- web session date
- user id
